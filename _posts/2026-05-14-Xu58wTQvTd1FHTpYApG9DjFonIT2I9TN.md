## Claude Code 上线 /goal 模式：AI编程从「生成代码」转向「闭环交付」

OpenAI、Anthropic、Hermes在11天内不约而同上线了「Ralph Loop」功能——让AI不完成任务绝不罢休。起因是澳洲牧羊大叔Geoffrey Huntley写的三行bash脚本，让Agent循环执行直到任务完成。

这意味着你给AI下单后，它会自己反复尝试直到搞定，而不是跑两轮就停下来问你。AI编程的核心战场正式从「生成代码」转向「闭环交付」。

## GitHub Copilot 6月改按量计费，Agent用户可能翻倍

6月1日起，Copilot从固定请求数改为token计费（1 credit = $0.01）。轻量用户基本不变，但频繁使用Agent模式跑自动化任务的人可能要多掏钱。代码补全和Next Edit建议仍然免费。

习惯让Copilot自主完成多个步骤长任务的人，下个月账单可能会涨。建议关注实际用量。

## Anthropic 估值1.2万亿超越 OpenAI，Claude Code配额翻倍

Anthropic在Pre-IPO市场隐含估值达1.2万亿，正式反超OpenAI。收入同比增长80倍，靠Claude Code的爆发式增长。同时宣布Claude Code每周使用配额提升50%，并与SpaceX合作新增30万千瓦算力。

Anthropic正在成为AI编程领域的强势选手。

## 百度 Create 2026：李彦宏称AI从模型竞争转向Agent竞争

李彦宏提出「自进化」概念——AI产品爆火靠的不再是模型本身，而是基于模型的Agent系统。同时发布秒哒3.0（支持直接生成APP）、DuMate通用Agent、Miaoda代码生成Agent等产品。

下一个赛道不是更牛的模型，而是「能把事干完」的Agent。学会构建Agent工作流比追新模型更重要。

## NVIDIA 官方支持 Hermes Agent + Qwen 本地部署

开源Agent框架Hermes上线5个月日处理2240亿token，超越OpenClaw成为全球最常用Agent。NVIDIA推出官方部署指南，支持RTX显卡和DGX Spark系统上本地运行，配合阿里Qwen 3.6模型。

Qwen 3.6（35B MoE架构，只需20GB内存）+ Hermes，本地跑AI Agent的新组合值得关注。
