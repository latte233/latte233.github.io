今天的AI圈热闹得像是过年。DeepSeek首轮融资估值破3500亿、国产大模型文心5.1登顶全球榜单、中国大模型调用量单周超越美国——这一天发生的事，足够写进中国AI发展史。

## 热点速递

### DeepSeek首轮融资估值515亿美元，刷新AI融资纪录

据证券时报今日报道，DeepSeek计划首轮募资最高500亿元，估值超过3500亿元（约515亿美元）。创始人梁文锋个人出资200亿元，通过直接与间接持股合计控制公司84.29%的股权。

DeepSeek一直被视为"不差钱"的代表——背靠幻方量化，从未有过资金压力。但梁文锋突然开放融资，核心目的是给员工期权一个市场定价。"在AI行业，人才密度基本决定了成败，股票期权通常是核心人才薪酬的重要甚至核心组成部分。"

与此同时，Kimi已完成20亿美元融资，阶跃星辰将完成近25亿美元融资。**"大模型五强"中四家身家已突破千亿元**，这一幕让人想起若干年前的"AI四小龙"盛景——旧人渐隐，新贵登台。

### 国产大模型历史性突破：文心5.1登顶全球第四

百度文心大模型5.1版本在LM Arena搜索榜中斩获国内第一、全球第四，也是目前唯一上榜的国产大模型。

更震撼的数据是：4月27日至5月3日这一周，**中国大模型调用量达到7.942万亿Token，环比暴涨81.7%，全球占比超过33%，再次超越美国**。

文心5.1采用"多维弹性预训练"技术，总参数压缩至约800B（前代1/3），预训练成本只有同规模的6%。这意味着中国AI技术正式跻身世界第一梯队。

### 三部门联合发文：智能体发展迎来国家顶层设计

国家互联网信息办公室、国家发展改革委、工业和信息化部联合印发《智能体规范应用与创新发展实施意见》。

文件明确了五大方向：**科学研究、产业发展、提振消费、民生福祉、社会治理**，并列出19个典型应用场景，重点布局先进制造、生物医药、量子信息、未来通信等领域。

构建覆盖智能体研发、上线、运行全生命周期的"备案—评估—应急处置"三层管理制度，精准适配智能体技术特性与发展规律。

### OpenAI连发大招：GPT-5.5 Trusted + 实时语音API

OpenAI今日连发两大更新：

1. **GPT-5.5 Trusted版本**：将高危场景下的幻觉率降至0.3%，医疗、金融等行业终于可以放心用AI了
2. **三款实时语音模型**：支持128K上下文、70种语言同声传译，多模态交互体验直接拉满

GPT-5.5 token效率比Claude少近40%，成本断崖式下降。Codex周下载暴涨1397%。但"缺乏人味儿"的交互体验仍是短板，Altman自己也调侃这个"自闭天才"标签。

与此同时，xAI正式开放自定义语音API，Cerebras上调IPO定价至150-160亿美元，博通获得阿波罗黑石350亿美元融资，订单积压已达730亿美元。

### 清华大学长文本技术突破：准确率99.3%

清华大学在长文本技术上实现重大突破——**千亿模型的长对话准确率达到99.3%**，成功打破了长期以来大模型在长上下文处理中的瓶颈。

这意味着大模型终于可以稳定地处理超长文档对话，不再出现"说到后面忘了前面"的尴尬。

## 早间资讯

### Hermes Agent悄悄登顶

OpenRouter上Hermes Agent的Token调用量超过了OpenClaw，日均2710亿。

核心是"自己教自己"——运行过程中自动生成Skill文件，边跑边优化。三层记忆（工作/情节/程序）让它越用越聪明。开发者正在用脚投票。

### AI写代码的Bug多1.7倍

CodeRabbit分析了470个开源项目，结论有点扎：AI参与代码的问题量比纯人类代码高1.7倍，逻辑错误涨75%，安全漏洞多了2.74倍。

说白了，AI在做"统计学模仿"，它不知道自己在写什么。局部看着像那么回事，全局一跑就出事。

有意思的是，报告给的解药是SDD——先让AI把规格写清楚，人审了再生成代码。别让AI对着模糊需求盲猜。

### 38万个AI应用在"裸奔"

RedAccess出了个报告：Lovable、Replit这些AI开发平台建的应用，大约38万个完全公开可访问，近2000个有明确数据泄露。医院员工信息、企业广告数据、客户联系方式全暴露在外面。更离谱的是部分应用连管理员权限都开着，删库就是点一下的事。

用AI工具搭东西的人越来越多，安全配置这种"boring stuff"反而成了最大的坑。

### Claude Code之父：我今年没手写一行代码

Boris Cherny在红杉AI峰会上说，去年10-11月模型就能写100%代码了，编程问题"已经解决了"。他现在主要在手机上干活，每天几十个PR。

"编程已解决"这话说得有点满，但"AI编排者"这个新角色确实在成型。开发者从写代码变成写意图、审结果、做决策。

### A2A + MCP成了标配

Google的A2A协议解决Agent间互通，Anthropic的MCP协议统一工具调用标准。2026年了，这俩已经成了AI Agent架构的两大支柱。

从单点工具调用升级到多Agent协同，"数字员工集群"不再是PPT上的概念。

## 午间资讯

### Claude Code AutoMode：AI编程进入无人值守

Anthropic推出Claude Code AutoMode，内置AI风险分类器，93%的操作可自主执行，只有批量删除、改生产配置这类高风险操作才需要人工确认。支持"一次需求→自动拆解→编写→调试→测试"全链路闭环。

实测20个任务完成17个，完成时间降到手动编码的25%。这不是小修小补，是AI编程工具的范式转变。

### 字节Trae Solo三端全量上线

iOS/Android App + Web + PC三端共享同一个Agent，支持跨端任务接力——手机发起，电脑执行，进度同步手机。内置飞书CLI、GitHub、定时任务集成。

AI编程从桌面专属正式进入移动场景，通勤路上也能提交PR了。

### SKILL.md接管Agent生态

SKILL.md把人类操作经验固化为机器可读SOP，通过metadata + 执行流程 + 异常处理三段式结构，把"一次性对话"变成"可复现、可移植的结构化技能包"。

这和SDD（规格驱动开发）高度相关——SKILL.md本质上就是Agent领域的规格驱动开发。

### 网易CodeWave企业级AI Coding平台

国内首个明确以"企业级可控交付"为核心的AI编程平台，NASL语言 + SDD + IDE三位一体架构，解决"Demo能跑，上线不敢"的问题。150个功能点项目从半月缩至数天。

SDD在国内企业级场景的首次正式落地。

### AMD炸裂：数据中心暴涨57%，HBM全年售罄

Q1营收102.5亿美元同比涨38%，数据中心同比涨57%。HBM内存全年产能售罄，美光净利润同比暴涨772%。另外AMD跟英特尔联手搞x86新指令集。

AI算力需求还在加速，HBM赛道继续躺着赚钱。
