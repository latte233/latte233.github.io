## 百度 Create 2026：Robin Li 力推「日活跃Agent」新指标

百度在 Create 2026 大会发布 Agent 全家桶——通用 Agent DuMate、编程 Agent 秒哒（App版+企业版）、数字人平台伊鸣升级、自进化 Agent Famou 2.0。

李彦宏首次提出用 DAA（Daily Active Agents）替代 Tokens 作为衡量AI价值的核心指标，预测全球 DAA 将突破100亿。

秒哒的「用嘴写代码」能力让「一次性软件」成为可能，开发成本趋近于零。对开发者来说这是个巨大的机会信号——也是巨大的危机信号。

## Apple Xcode 26.5：AI编程体验终于不憋屈了

两个关键更新：AI助手支持消息队列排队（不用等一个回复完再发下一个），Coding Agent 能主动询问澄清问题再动手，减少改错方向的风险。

这两个改进直接解决了之前AI编程最大的痛点——要么干等，要么改错方向。Apple 也在认真做 AI Coding 了，Mac 开发者是受益者。

## UiPath：企业级 Coding Agents 原生集成

UiPath 成为首个在企业自动化平台中原生集成 Claude Code 和 OpenAI Codex 的工具，内置治理、审计、RBAC权限控制。企业可跨部门使用不同Agent，不用操心合规问题。

企业AI落地最难的就是治理和合规，UiPath把这个痛点解决了。非技术用户也能用自然语言驱动自动化——企业级AI Agent市场正式打开。

## UnqorkAI：AI生成代码的治理平台来了

Unqork 发布 UnqorkAI，专注解决AI生成代码的「代码 sprawl」问题——让AI生成的应用也具备企业级的安全性、可维护性和治理能力。前IBM Watson负责人、首席AI官亲自操刀。

AI编程普及后，代码维护成本是下一个大坑。这个方向的企业服务需求会爆发。

## Vibe Coding 概念持续升温：百度秒哒领跑国内

IT之家发布5月AI Coding工具榜单，百度秒哒凭借「多Agent协作+自然语言生成完整应用」领跑国内，美团NoCode、扣子等也在快速迭代。Vibe Coding 正在从概念走向实用。

Vibe Coding = 用自然语言描述需求，AI帮你生成完整应用。门槛低、速度快，适合快速验证想法、做MVP。
