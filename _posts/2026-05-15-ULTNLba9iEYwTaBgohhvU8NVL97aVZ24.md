## OpenAI Codex 登陆手机

Codex 现已集成到 ChatGPT App，开发者可在手机上管理代码任务、审核输出、切换模型。支持多线程管理，5月14日已预览上线。

编程工具从桌面拓展到移动端，随时改代码成现实了。

## 微软将停用内部 Claude Code

6月30日前取消 Experiences + Devices 部门的 Claude Code 许可，推动全员迁移至 GitHub Copilot CLI。涉及 Windows、Office、Teams 等数千名工程师。

企业级工具选型风向标，Copilot 生态加速扩张。

## GPT-5.5 悄默更新，编程能力再破纪录

OpenAI 更新 GPT-5.5，拿下 gramBench 等多个编程榜单第一。自主完成架构设计、漏洞修复能力突破62%。

初级编码岗位继续贬值，程序员核心能力转向架构与AI调度。

## 微软发布 Microsoft Agent Framework（开源）

统一的多Agent框架，支持MCP、A2A、OpenAPI三大协议，无缝衔接从研究到生产的流程。融合 Semantic Kernel 企业能力与 AutoGen 编排模式。

企业级Agent开发门槛降低，生态互操作性成新标准。

## DeepSeek 开放识图模式

灰度测试「视觉原语思考」框架，处理800×800图片仅消耗约90 tokens，远低于GPT等主流模型。支持独立纯视觉理解，不联动搜索。

多模态成本革命，边缘部署可行性大增。

## specdd 上线：SDD变成AI coding skill

把规格驱动开发变成一个AI coding skill，直接丢进Claude/Codex/Cursor用。spec-driven开发工作流正在从重型框架向轻量化skill演进。
