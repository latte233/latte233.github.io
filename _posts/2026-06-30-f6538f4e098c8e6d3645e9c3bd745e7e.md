## 本期速览

本周 AI 行业进入"多线爆发"模式。6月26日 OpenAI 发布 [GPT-5.6](https://openai.com/index/gpt-5-6/) 系列（Sol/Terra/Luna 三级架构），Sol 在 TerminalBench 2.1 以 91.9% 登顶，标志着 OpenAI 正式回应 Claude Fable 5 的挑战。同日 Anthropic 指控阿里巴巴通过 2.5 万假账号、2880 万次交互蒸馏 Claude 模型能力，这是已知最大规模的模型蒸馏攻击事件，AI 知识产权保护问题首次升级到国家层面。商业格局方面，SpaceX 以 600 亿美元全股票收购 Cursor 母公司 Anysphere，Cursor 同步发布 1.5 万亿参数自研模型和 Git 平台 Origin，正面挑战 GitHub——AI Coding 工具市场进入巨头混战阶段。国内方面，字节跳动 6月23日发布豆包 Seed 2.1 Pro，Coding 能力进入国际第一梯队。GitHub Copilot 6月1日正式切换 Token 计费（AI Credits），部分开发者月账单从 $39 飙升至 $750+，引发大规模迁移讨论。

## 重点信号详解

### GPT-5.6 Sol/Terra/Luna 发布

**象限**：Models & Research
**环位**：Trial

**是什么**：OpenAI 6月26日发布 GPT-5.6 系列，采用全新三级命名体系。Sol（旗舰）专攻编程和科学推理，Terra（均衡）定位日常工作达 GPT-5.5 水平但价格减半，Luna（轻量）面向高吞吐场景。API 定价：Sol $5/$30，Terra $2.50/$15，Luna $1/$6（每百万 Token）。

**为什么重要**：Sol 在 TerminalBench 2.1 以 91.9% 超越 Claude Mythos 5（CTF 命中率 96.7%），重新确立 OpenAI 在编程推理领域的领先地位。三级定价体系让开发者可以按场景精细化选型，Terra 作为 GPT-5.5 平替方案成本直降 50%。

**当前状态**：有限预览阶段，仅约 20 家受信合作伙伴可通过 API 和 Codex 访问。受美国政府审查，全面开放时间待定。尚未整合至 ChatGPT 供普通用户使用。

**建议下一步**：关注 Sol 全面开放后的实际 Coding 表现对比（vs Claude Fable 5），Terra 可作为 GPT-5.5 的降本替代方案纳入评估。短期无需行动，等待开放后做 A/B 对比。

### SpaceX 600 亿美元收购 Cursor

**象限**：Practices & Ecosystem
**环位**：Watch（交易 Q3 完成前仅做观察）

**是什么**：6月16日，SpaceX 宣布以 600 亿美元全股票交易收购 Cursor 母公司 Anysphere，是 AI 开发者工具领域史上最大收购。同日 Cursor 在 Compile 26 大会发布 1.5 万亿参数自研模型和 Git 平台 Origin。

**为什么重要**：这笔交易重塑了 AI Coding 市场格局。Cursor（ARR $4B 但市场份额从 41% 降至 26%）将获得 SpaceX/xAI 的海量算力和 Grok 生态支持，但也面临与 xAI 整合的不确定性。Origin 直接挑战 GitHub，Agent-first 代码托管可能成为新范式。

**当前状态**：协议已签，Q3 2026 预计完成。Cursor 现有股东将获得 SpaceX A 类普通股。Origin 秋季上线，1.5T 模型计划年内发布。

**建议下一步**：暂无直接行动需求。密切关注 Cursor 产品路线是否因收购发生偏移，以及 Origin 平台是否会分流 GitHub 的开发者生态。

### Anthropic 指控阿里巴巴蒸馏 Claude

**象限**：Practices & Ecosystem
**环位**：Assess

**是什么**：6月24日 CNBC 披露，Anthropic 于 6月10日致函美国参议院，指控阿里巴巴 Qwen 实验室关联运营者使用约 2.5 万假账号、在 4月22日至 6月5日间与 Claude 模型进行 2880 万次交互，实施"已知最大规模蒸馏攻击"。

**为什么重要**：这是 AI 模型知识产权保护问题首次升级到国家立法层面。如果指控成立，将深刻影响中美 AI 竞争格局、API 服务商的安全策略、以及开源与闭源模型之间的信任边界。同时引发讨论——Anthropic 自身也曾面临"数据抓取"争议，标准如何一致适用？

**当前状态**：阿里巴巴尚未公开回应。Anthropic 已在 6月12日对其 Fable 5 和 Mythos 5 模型实施地域访问限制作为防御措施。事件仍在发酵中。

**建议下一步**：关注事件后续发展对 API 使用条款和模型访问策略的影响。如果团队使用 Claude API，建议审查内部使用规范，确保合规。

### GitHub Copilot 切换 Token 计费（AI Credits）

**象限**：Practices & Ecosystem
**环位**：Trial

**是什么**：6月1日起，GitHub Copilot 全面从 Premium Request 计费切换为 Token-based AI Credits 计费。每个计划分配月度 Credits 额度，超出按用量付费。部分使用 Agent 功能的开发者报告月费从 $39 跳升至 $750。

**为什么重要**：这一定价模式变更直接影响全球数百万 Copilot 用户的使用习惯和成本预期。Token 计费让重度 Agent 用户成本飙升 10-50 倍，已引发大规模向 Claude Code 等竞品迁移的讨论。这也反映了 AI Coding 工具从"补全"到"Agent"的转变所带来的商业模式重构压力。

**当前状态**：已全面生效。社区反响强烈，特别是使用多步 Agent 工作流的用户受影响最大。Claude Code 以 $20/$100 月度定额制成为直接受益者。

**建议下一步**：评估团队 Copilot 使用成本变化，特别是使用 Agent 模式的场景。如果成本显著上升，建议对比 Claude Code 的性价比。
