## Claude Code AutoMode：AI编程进入无人值守

Anthropic 推出 Claude Code AutoMode，内置AI风险分类器，93%的操作可自主执行，只有批量删除、改生产配置这类高风险操作才需要人工确认。支持"一次需求→自动拆解→编写→调试→测试"全链路闭环。

Boris Cherny（Claude Code之父）已经100%用AI写代码了。

## 字节 Trae Solo 三端全量上线

iOS/Android App + Web + PC 三端共享同一个Agent，支持跨端任务接力——手机发起，电脑执行，进度同步手机。内置飞书CLI、GitHub、定时任务集成。

AI编程从桌面专属正式进入移动场景，通勤路上也能提交PR了。

## GPT-5.5的"自闭天才"标签

GPT-5.5 token效率比Claude少近40%，成本断崖式下降。Codex周下载暴涨1397%。但"缺乏人味儿"的交互体验仍是短板，Altman自己也调侃这个"自闭天才"标签。

AI编程工具的竞争正在从"模型智商"转向"综合成本效率"，开发者体验成了新战场。

## SKILL.md接管Agent生态

SKILL.md把人类操作经验固化为机器可读SOP，通过metadata + 执行流程 + 异常处理三段式结构，把"一次性对话"变成"可复现、可移植的结构化技能包"。可在Claude Code / Cursor / OpenClaw三端通用。

这和SDD（规格驱动开发）高度相关——SKILL.md本质上就是Agent领域的规格驱动开发。

## 网易CodeWave企业级AI Coding平台

国内首个明确以"企业级可控交付"为核心的AI编程平台，NASL语言 + SDD + IDE三位一体架构，解决"Demo能跑，上线不敢"的问题。150个功能点项目从半月缩至数天。

SDD在国内企业级场景的首次正式落地，网易这个案例值得研究。
