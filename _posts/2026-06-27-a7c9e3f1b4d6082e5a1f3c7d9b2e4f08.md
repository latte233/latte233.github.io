## 重点信号详解

### Superpowers — Agent Skill 标准框架

**象限**：Frameworks & Infrastructure
**环位**：Adopt

**是什么**：Superpowers 定义了 AI Agent Skill 的标准格式，提供 50+ 预置技能（GitHub 操作、代码审查、项目规划、深度研究等），支持 Claude Code、OpenAI Codex 等主流 Agent 平台。

**为什么重要**：Agent 之间的差距不在模型，在"技能"。Superpowers 正在成为 Agent Skill 生态的事实标准，就像 npm 之于 Node.js、VSCode Extensions 之于编辑器。6 月上半旬 Claude Skills 模块化生态成为 GitHub 最大爆点，多个基于 Superpowers 的 Skill 项目（last30days-skill、academic-research-skills、taste-skill 等）单周新增数千甚至上万 Star。

**当前状态**：21.5 万 Star，社区极其活跃，已被 Claude Code 和 Codex CLI 原生支持。技能覆盖面广，从代码开发到内容创作到项目管理。

**建议下一步**：立即安装并使用。重点关注与团队工作流匹配的 Skill，如 GitHub Ops、Code Review、深度研究等。

### GitHub Copilot 多智能体编排平台升级

**象限**：Tools & Products
**环位**：Trial

**是什么**：2026 年 6 月 2 日，GitHub Copilot 迎来史上最大规模升级——发布独立桌面应用（Desktop App）、SDK 正式 GA 支持 6 种语言、Sandboxes 提供云/本地双重隔离执行、CLI 新增橡皮鸭代码审查和语音输入，并集成 Claude Fable 5 和 OpenAI Codex 模型。从 IDE 插件进化为多智能体编排平台。

**为什么重要**：GitHub Copilot 从"代码补全工具"跳到了"自主 Agent 平台"。这意味着微软/GitHub 不再把 AI 编程定位为辅助，而是定位为"AI 替你跑整个开发流程"。SDK GA 让第三方开发者可以构建自己的 Copilot 扩展，生态想象空间巨大。

**当前状态**：已发布并可用。Desktop App + SDK + Sandboxes 三件套齐备。注意 Claude Fable 5 在 6 月 12 日被暂停接入（Anthropic 侧原因）。

**建议下一步**：已在使用 Copilot 的团队建议立即升级体验新功能，特别是 Sandboxes 和 SDK。正在评估 AI Coding 工具的团队将此作为 Cursor / Claude Code 之外的重要候选。

### Cursor 3.8 发布 /automate 技能

**象限**：Tools & Products
**环位**：Trial

**是什么**：Cursor 3.8 于 6 月 18 日发布，核心新功能是 `/automate` 技能——支持 Slack 和 GitHub 触发器的自动化工作流，以及 Computer-Use 功能让云端 Agent 可以操控桌面界面。

**为什么重要**：Cursor 从"AI IDE"进化为"AI 自动化平台"。开发者可以设置触发条件（Slack 消息、GitHub PR 事件等），让 Cursor Agent 自动执行编码任务。这是 AI Coding 工具从"被动辅助"到"主动执行"的关键一步。

**当前状态**：已发布。/automate 和 Computer-Use 为新功能，社区反馈积极但尚需更多生产环境验证。

**建议下一步**：Cursor 用户立即升级到 3.8 体验 /automate。可考虑在低风险场景（如自动 Code Review、自动生成测试）中试用触发器功能。

### 开源模型"五国杀"

**象限**：Models & Research
**环位**：Trial（Qwen 3.6、DeepSeek V4）/ Assess（Llama 4、GLM-5.1、Gemma 4）

**是什么**：2026 年开源模型进入白热化竞争——阿里 Qwen 3.6、Meta Llama 4、DeepSeek V4、Google Gemma 4、智谱 GLM-5.1 五大阵营同台竞技。核心趋势：MoE 架构成为标配，开源模型在编码和推理上逼近闭源前沿。

**为什么重要**：Qwen 3.6-35B-A3B 的稀疏比最激进（35B 参数只激活 3B），可以在单个消费级 GPU 上运行且性能不输参数量大 10 倍的模型。DeepSeek V4-Pro 在 LiveCodeBench 上创下 93.5% 的开源新纪录。开源与闭源的差距正在快速缩小到个位数百分点。

**当前状态**：五款模型均已发布并可通过 HuggingFace 下载。Qwen 3.6 和 DeepSeek V4 在性价比和可部署性上最具优势。

**建议下一步**：如果团队有本地部署/微调需求，优先评估 Qwen 3.6-35B-A3B（消费级硬件友好）和 DeepSeek V4（编码性能最强）。
