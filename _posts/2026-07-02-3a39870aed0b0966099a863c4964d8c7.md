## 本期速览

本周最重大的事件是 GPT-5.6 系列在经历 4 天受限预览后于 6 月 30 日正式全面开放，Sol/Terra/Luna 三档模型同步登陆 ChatGPT、API 和 Azure OpenAI，Cerebras 推理加速（750 token/s）同步上线。这意味着上期报告中"受限预览"的状态已结束，开发者可以立即评估 Sol 与现有 Claude Fable 5 的能力差异和成本对比。Anthropic 同样在 6 月 9 日发布了 Claude Fable 5——首个面向公众开放的 Mythos 级模型，SWE-Pro 基准 80.3% 碾压全场，定价 $10/$50 每百万 Token。两大顶级模型同期开放，AI 模型性能天花板本周正式刷新。

AI Coding 工具格局本周发生重大变动：Windsurf 于 6 月 2 日正式更名为 Devin Desktop，Cascade Agent 7 月 1 日 EOL，Rust 重写的 Devin Local 接替；Google Gemini CLI 于 6 月 18 日停服，10 万 Star 的开源项目被转为闭源 Antigravity CLI，引发开源社区强烈反弹。OpenAI Codex 同步推进旧模型退役潮——GPT-5.2 和 GPT-5.3-Codex 6 月 30 日起停止新请求。AI Coding 赛道正经历有史以来最剧烈的洗牌。

AI Engineer World's Fair 2026 于 6/29-7/2 在旧金山举行，6000+ 工程师出席，是本年度最大 AI 工程会议。

### 巨头入场

**Google — Antigravity CLI**（5月20日 I/O 发布，6月18日生效）🏢

Google 在 I/O 2026 上发布 Antigravity CLI 2.0（agent-first 开发平台），同时宣布 Gemini CLI 于 6月18日停服。这是 Google 将 AI CLI 工具从开源转向闭源商业的重大战略转向。Antigravity CLI 采用 Chrome DevTools Protocol 和沙箱安全机制的全新架构，支持异步 Agent 后台运行——但闭源性质和对 6000+ 社区贡献的无视引发强烈批评。

**OpenAI — Codex Micro 硬件**（7月15日发售）🏢

OpenAI 与 Work Louder 联合推出 Codex Micro 可编程开发者宏键盘，首次以硬件形态进入开发者工具市场。这是 AI 编程工具从纯软件向硬件+软件融合发展的信号。

## 重点信号详解

### GPT-5.6 系列全面开放

**象限**：Models & Research
**环位**：Trial → 接近 Adopt

**是什么**：OpenAI 6月30日宣布 GPT-5.6 系列结束受限预览，Sol/Terra/Luna 三档模型正式向全部开发者开放。Cerebras 推理加速（750 token/s）同步上线。Sol 在 TerminalBench 2.1 以 91.9% 登顶，CTF 命中率 96.7%。

**为什么重要**：从上期的"仅约20家受信伙伴可访问"到本周全面开放，GPT-5.6 Sol 正式成为所有开发者可用的最强编程模型。Terra 作为 GPT-5.5 平替（性能相当、价格减半）立即可用于降本。三档分层定价（Sol $5/$30、Terra $2.50/$15、Luna $1/$6）与 Claude Fable 5（$10/$50）形成直接价格竞争。

**当前状态**：全面开放，ChatGPT/API/Azure 三端可用。Cerebras 加速版初期向部分客户开放。

**建议下一步**：立即评估 Terra 作为现有 GPT-5.5 工作流的替代方案（成本降50%）。对 Coding 重度场景，对比 Sol vs Claude Fable 5 的实际表现差异。

### Claude Fable 5 公开发布

**象限**：Models & Research
**环位**：Adopt

**是什么**：Anthropic 6月9日发布的首个面向公众开放的 Mythos 级模型。SWE-Pro 编程基准 80.3%（领先第二名11个百分点），支持 1M 上下文窗口、自适应思考（Adaptive Thinking），在软件工程、知识工作、视觉理解和科研方向全面超越 Opus 4.8。

**为什么重要**：Fable 5 是当前公开可用模型中编程能力的绝对天花板。Stripe 用它一天完成 5000 万行 Ruby 代码库迁移，蛋白设计速度提升 10 倍。对需要长周期自主编程（多天连续 session）的场景直接可用。

**当前状态**：Pro/Max/Team/Enterprise 计划可用（6/9-6/22 包含在订阅中，6/23 起需使用积分）。API 定价 $10/$50。Claude Code 已原生支持。

**建议下一步**：将 Fable 5 作为默认 Claude Code 模型用于复杂重构和大规模迁移任务。对比 Opus 4.8 在日常编程任务中的成本效益比。

### Windsurf → Devin Desktop 品牌重塑

**象限**：Tools & Products
**环位**：Trial

**是什么**：6月2日，Cognition 通过 OTA 更新将 Windsurf 正式更名为 Devin Desktop。Cascade Agent 于 7月1日 EOL，由 Rust 重写的 Devin Local 接替。新平台支持开源 Agent 客户端协议（ACP），允许 Codex、Claude Agent 等第三方 Agent 在同一编辑器中运行。

**为什么重要**：Windsurf 从 AI-first IDE 转型为多 Agent 管理平台，本质上是"IDE 正在变成 Agent 管理器"这一趋势的具象化。ACP 开放协议可能成为未来 Agent IDE 集成的标准。同时 ARR 8200 万美元、估值 260 亿，证明 AI Coding 工具的商业化已经成熟。

**当前状态**：已完成更名和迁移。Devin Local 替代 Cascade。ACP 协议开源。

**建议下一步**：如果团队使用 Windsurf，确认 Cascade 到 Devin Local 的迁移已完成（7/1 是 Cascade EOL 截止日）。评估 ACP 协议对自有 Agent 集成的价值。

### Gemini CLI 停服与开源信任危机

**象限**：Practices & Ecosystem
**环位**：Assess

**是什么**：Google 于 6月18日正式停止 Gemini CLI 免费/Pro/Ultra 用户服务，转向闭源的 Antigravity CLI。Gemini CLI 拥有 100K+ GitHub Star、6000+ 社区贡献 PR、Apache 2.0 协议——这些全部被废弃。

**为什么重要**：这是 2026 年最大的"开源转闭源"事件。Google 接受了 6000+ 社区贡献后将工具转为闭源商业产品，被广泛称为"钓鱼式商业模式"（bait-and-switch）。对依赖开源 AI CLI 工具的团队，这是一个重要的供应链风险警示——即便是大公司的开源项目也可能随时关闭。

**当前状态**：Gemini CLI 已停服。Antigravity CLI 1.0.1 可用但闭源。社区大量用户迁移至 OpenCode、Claude Code 等替代方案。

**建议下一步**：如果曾依赖 Gemini CLI，评估 OpenCode（完全开源）或 Claude Code 作为替代。审视团队工具链中其他大公司开源项目的关闭风险。
